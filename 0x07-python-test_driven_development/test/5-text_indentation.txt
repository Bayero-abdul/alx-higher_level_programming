The text_indentation examples
===========================

``text_indentation()``  prints a text with 2 new lines
after each of these characters: ., ? and :

Examples

	>>> text_indentation = __import__('5-text_indentation').text_indentation
	>>> text_indentation()
	Traceback (most recent call last):
		...
	TypeError: text_indentation() missing 1 required positional arguments: 'text'

	>>> text_indentation("")
	>>> text_indentation(".") #doctest: +NORMALIZE_WHITESPACE
	
	
	>>> text_indentation(".?:") #doctest: +NORMALIZE_WHITESPACE
	
	
	
	
	
	
	
	>>> text_indentation(234)
	Traceback (most recent call last):
		...
	TypeError: text must be a string
